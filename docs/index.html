Features

메모

메모를 작성하고 복사하여 관리합니다.

캘린더

날짜별로 메모를 확인합니다.

루틴

반복되는 일정을 루틴으로 등록하고 캘린더에서 노출합니다.

Google 로그인

Google 계정으로 간편하게 로그인합니다.

동기화

Supabase를 통해 기기 간 데이터를 동기화합니다.

오프라인 모드

오프라인 환경에서도 계정 상태를 유지합니다.

날씨

현재 위치(GPS) 기반으로 날씨 정보를 확인합니다.

태그

메모에 태그를 추가하여 분류합니다.

공휴일

공휴일 정보를 캘린더에서 확인합니다.

황금연휴 찾기

연차를 활용한 황금연휴를 찾아줍니다.

태그 필터

태그별로 메모를 필터링합니다.

Project Guard

모듈 간 의존성 규칙을 확인합니다.
